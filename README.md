# Alakazam

Alakazam is a simple AI chatbot built with Python, Flask, HTML, CSS, and JavaScript. It uses the DeepSeek ( deepseek-ai/deepseek-v4-flash ) model to generate intelligent, real-time responses through API integration.

## Features

* Real-time AI conversations
* Conversation memory for contextual responses
* Markdown support for formatted output
* Clean and responsive web interface
* Enter key support for sending messages
* Frontend and backend communication using the Fetch API

## Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* DeepSeek (`deepseek-ai/deepseek-v4-flash`)
* OpenAI-compatible API
* JSON

## Future Improvements
